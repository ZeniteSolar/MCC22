
# Setting up the environment

### 1. Virtual Environment

```bash
python3.10 -m venv .venv
source .venv/bin/activate
python -m pip --upgrade pip
```

### 2. Project requirements
```bash
python -m pip install -r requirements.txt
```

Open `tllocalmgr` and:
```bash
install circuitikz
exit
```

and then:
```bash
sudo texhash
```
